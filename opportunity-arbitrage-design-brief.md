# Opportunity Arbitrage — Visual System Brief

## Goal
Create a sharp, premium, operator-style visual language for the playbook and bundle assets.

Design should feel:
- high-signal
- tactical
- editorial
- slightly severe
- not startup-cute
- not corporate-consulting bland

## Core visual direction
- Black / off-white / muted sand / dark green / restrained red accents
- Strong typography hierarchy
- Clean grid
- Light technical / field-manual energy
- Dense but scannable

## Primary components to design

### 1. Five Filters Scorecard Spread
Purpose:
- one-page or two-page designed spread
- printable and screenshot-worthy
- should feel like a real operator tool, not worksheet clipart

Must include:
- title: Five Filters Scorecard
- six scoring rows or cards
- total score area
- interpretation band area
- room for short notes

Visual cues:
- boxy modular layout
- score cells 1–5
- strong contrast
- subtle icons per filter if useful

### 2. Decision Icons
Need 3 simple icons used consistently:
- Kill = skull
- Build = hammer
- Backlog = parking sign

Style:
- bold monochrome
- editorial / stencil / signal icon feel
- readable small and large
- should work in print and digital

### 3. Opportunity Library Card Template
Purpose:
- reusable visual template for Part IV wedges
- reader should scan archetypes quickly

Must include fields:
- opportunity name
- buyer
- pain
- trigger
- wedge
- why it works
- category / archetype tag

Visual cues:
- colored top band by archetype
- compact metadata panel
- clean left-to-right scan path
- could support icon tag for archetype

## Archetype colors (proposed)
- Compliance = dark red
- Margin / Tracking = brass / amber
- Workflow = dark green
- Lead Gen = steel blue
- Searchable Intelligence = charcoal / violet-gray

## Deliverables wanted now
1. one scorecard spread concept
2. one icon strip with kill/build/backlog
3. one opportunity card template concept
4. one sample filled card for a wedge like SauceStack or Permit Intelligence
