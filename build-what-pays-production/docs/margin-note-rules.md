# Margin Note Rules — Build What Pays

Use margin notes sparingly. They should improve scan value, not become a second essay.

## Use margin notes for
- definitions
- warnings
- score thresholds
- operator translations
- tactical examples
- appendix/tool cross-references

## Avoid
- repeating the body text in softer language
- introducing a whole new argument
- long notes that feel like mini-paragraphs
- more than 1–3 margin notes per spread

## Preferred note types
### Definition
Short term definition.

### Warning
A hard caution or self-deception checkpoint.

### In Practice
A concise real-world application.

### Score Reminder
Short threshold such as:
- 26+ = strong build candidate
- 2+ major kill signals = stop

## Tone
- compressed
- tactical
- verdict-like
- no fluff

## Test
If removing the note does not reduce clarity, scan value, or usefulness, cut it.
