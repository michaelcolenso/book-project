# Dirty Data Directory

A starter list of signal-rich source categories where opportunity arbitrage often lives.

## Categories
- Permits
- Licenses
- Inspections
- Complaints
- Recalls
- Filings
- Violations
- Registrations
- Procurement records
- Public performance databases
- Enforcement actions
- Court and docket-adjacent public records
- Business registrations and amendments
- Labor and wage records
- Public health and safety datasets

## Good hunting pattern
For each source, ask:
1. Who cares about this?
2. What trigger does it create?
3. Is the interface ugly?
4. Is the buyer already paying for a weaker version of this signal?
5. Can I ship the first useful layer as an alert, report, database, or narrow tool?

## Example source families
- City / county building permit feeds
- State contractor license boards
- State ABC / liquor licensing boards
- FDA / USDA / health inspection sources
- CFPB complaint datasets
- Department of Labor enforcement records
- SAM.gov and procurement feeds
- UCC filing databases
- Secretary of State business filings
- Federal and state recall datasets
- Public inspection and enforcement archives

## Warning
Dirty data is only valuable if the buyer, trigger, and use case are clear. Messy data alone is not an opportunity.
