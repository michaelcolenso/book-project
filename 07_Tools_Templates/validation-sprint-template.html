Opportunity Arbitrage Toolkit

30-Day Validation Sprint

Use this to turn one opportunity into a Build / Backlog / Kill decision in 30 days.
Opportunity

Buyer

Wedge

Week 1 — Verify the Pain

Objective
Confirm the pain is real, repeated, and costly enough to matter.
Tasks
• Collect complaint threads
• Collect money-spent statements
• Collect ugly workaround examples
• List trigger moments
• Write one-page opportunity memo

Week 2 — Test the Offer

Objective
See whether the buyer responds to a narrow promise.
Tasks
• Write one-line offer
• Create simple landing page / DM / email
• Send outreach
• Track replies and objections

Week 3 — Deliver Manually

Objective
Deliver the result manually and learn what buyers actually value.
Tasks
• Deliver report / checklist / alert / audit
• Track useful inputs and outputs
• Note what can be simplified
• Identify repeat-use behavior

Week 4 — Decide Brutally

Objective
Assign a verdict based on evidence, not attachment.
Questions
• Did I find real pain?
• Did real buyers respond?
• Did anyone pay or commit seriously?
• Did I learn enough to define v1?
• Is this stronger than the next-best option?

Green Light — Build

Someone wants this repeatedly. Buyers engage seriously. The manual version proves the wedge.

Yellow Light — Pivot

The signal is real but the shape is wrong. Sell the report, checklist, or service before software.

Red Light — Kill

No real buyer motion. No meaningful responses. No evidence strong enough to earn the next 30 days.
